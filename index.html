Transparent Object Dataset for Human-Robot Interaction

A dataset for transparent object detection, classification, and surface reconstruction captured using a humanoid robot.

Dataset Overview

This dataset consists of 7700 frames captured from 100 scenes involving transparent objects. It includes data from multiple cameras mounted on a humanoid robot. The dataset is designed for challenging tasks such as object detection, depth estimation, and robot grasping for transparent materials like glasses.
